# 🛍️ Oyinkansola Closet Web Project

A modern **fashion e-commerce website** designed to showcase clothing products in a clean and responsive interface.
This project focuses on delivering an engaging user experience with a visually appealing layout suitable for online fashion stores.

---

## ✨ Features

* 🛒 Product showcase with stylish layout
* 📱 Fully responsive design (mobile, tablet, desktop)
* 🎨 Modern and clean user interface
* 🧭 Easy navigation for browsing products
* ⚡ Fast loading pages
* 🖼️ Image-based product catalog
* 💻 Cross-browser compatibility

---

## 🧰 Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and responsive design
* **JavaScript** – Interactive features
* **Git** – Version control
* **GitHub** – Project hosting and collaboration
